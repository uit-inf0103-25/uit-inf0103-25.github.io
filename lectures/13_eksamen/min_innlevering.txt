Oppgave 3

Jeg forstår det som at A holder nøyaktig n tekststrenger og ikke termineres av
noen nullpeker

a)
blir å trenge en funksjon som finner lengden på en tekststreng, skriver det som hjelpefunksjon under:

// antar alle strengene er nullterminert
int lengde(char *str) {
  int len = 0;
  while(str != 0) {
    len++;
    str++;
  }

  return len;
}

for lengste kan jeg bruke min hjelpefunksjon

// returnerer index i a til lengste streng
int lengste(char **A, int n) {
  int lengst = 0;
  for (int i = 0; i < n; i++) {
    if (lengde(A[i] > lengden[lengst]) lengst = i;
  }

  return i;
}
