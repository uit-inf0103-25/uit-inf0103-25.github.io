# Eksamenstips: Skaff deg oversikt og bruk tiden fornuftig

Les alle oppgavene før du starter. Gjør det enkleste først. 

# Eksamenstips: kommuniser med sensor

* Skriv ned hva dine antakelser om oppgaven er.
* Forklar hvordan du tenker

Hvis noe i oppgaveteksten virker tvetydig eller åpent for tolkning kan du
skrive noe slikt som:

> Dette kan etter min mening enten bety X eller Y, jeg velger å tolke det som
> om det er Y som gjelder.

Er du usikker på hvordan du skal implementere noe kan det være verd å
forklare din forståelse oppgaven og hvordan du tror en løsning ser ut.

# Eksamenstips: du blir ikke straffet for å ta feil
Selv om du er usikker på svaret på en oppgave har du ingenting å tape på å
gjette.
